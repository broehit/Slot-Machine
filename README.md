# Slot Machine

A vibrant, web-based slot machine game featuring neon aesthetics and classic slot mechanics.

[**Play the Live Demo**](https://broehit.github.io/Slot-Machine/)

## Features

- **Slot Mechanics**: Classic 3-reel slot machine experience.
- **Betting System**: Adjustable lines (1-3) and customizable bet amounts per line.
- **Economy**: Start with a balance, deposit funds, and track your winnings.
- **Visuals**: Neon-themed design with smooth animations.
- **Responsive**: Works on desktop and mobile devices.

## How to Play

1. **Deposit**: Enter an amount and click "DEPOSIT" to add funds to your balance.
2. **Set Bet**: Choose the number of lines (1-3) and your bet per line.
3. **Spin**: Click "SPIN" to start the reels.
4. **Win**: Match symbols across your active paylines to win!

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Styling, Flexbox/Grid, and animations.
- **JavaScript (Vanilla)**: Game logic and DOM manipulation.

## Setup

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/broehit/Slot-Machine.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Slot-Machine
   ```
3. Open `index.html` in your browser.
